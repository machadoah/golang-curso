## 🛠️ Comandos Úteis

### Inicializar um novo módulo Go

```bash
go mod init nome-do-modulo
```



### Importar pacotes externos

```bash
go get caminho/do/pacote
```



### Remover dependências não utilizadas

```bash
go mod tidy
```
